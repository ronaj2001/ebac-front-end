formação Front-End

Desenvolva sites funcionais e de alto impacto visual.

O que você vai aprender quando fizer esse curso:

• HTML • CSS • Design responsivo • JavaScript

Seja desenvolvedor de Front-End e desenvolva sites funcionais e de alto impacto visual.

O que você vai aprender quando fizer esse curso:

• HTML • CSS• Design responsivo • JavaScript

Sobre o curso

Forme estruturas de sites dinâmicos e inove com o HTML5 e CSS3! Crie interatividade, estruturas e biblioteca com o JavaScript, além de gerenciar metodologias síncronas e assíncronas e claro: Seu site será completamente responsivo.

Como funciona o curso?

O curso é 100% online. Isso significa que você escolhe a hora e o lugar de estudar. E quando tiver dúvidas, nossos tutores te respondem em até 5 minutos, de forma personalizada.

Conteúdo

Módulo 1Fundamentos Web e HTML

  • Identifique o funcionamento do navegador como ferramenta para interpretar sites da web.
  • Utilize estruturas de tags para a web.
  • Use objetos multimídia nos sites da web.
  • Utilize formatos de mídia para a web.
  • Aplique metodologias para incorporar gráficos vetoriais (SVG).

Módulo 2CSS e Web Design Responsivo

  • Identifique o funcionamento do fluxo de elementos na estrutura HTML.
  • Aplique o padrão CSS.
  • Aplique propriedades de forma à estrutura.
  • Utilize propriedades de transformação e efeitos especiais.
  • Aplique propriedades de animação da estrutura.

Módulo 3Frameworks e Bibliotecas CSS

  • Defina recursos de design para interfaces de usuários.
  • Utilize dos recursos de internacionalização e design.
  • Crie uma barra de ferramentas de compilação para facilitar a experiência do usuário.
  • Desenvolva menus usando tipos de navegação.

Módulo 4 Desenvolvimento Web com JavaScript

  • Entenda o que é JavaScript e como ele funciona.
  • Aprenda os fundamentos básicos de programação.
  • Conheça e empregue os seletores básicos JavaScript.
  • Depure o código JavaScript com o navegador.
  • Aprenda a usar o formato JSON e as APIs do HTML5 para dar uma maior interatividade.

Módulo 5 Frameworks e Bibliotecas do JavaScript

  • Identifique o que é uma estrutura JavaScript.
  • Saiba distinguir o que é uma biblioteca de JavaScript.
  • Diferencie estruturas e bibliotecas de JavaScript.
  • Identifique o que é jQuery e como funciona.
  • Reconheça o que é Protótipo e como funciona.
